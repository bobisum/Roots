Fast fully responsive admin theme, with everything you really need.

MAIN FEATURES

A fixed-fluid fully responsive layout admin theme, with everything you really need.
  • Clean HTML5
  • Easy to read and edit CSS3
  • SCSS files included in the theme files, making it easy to edit.
  • Breadcrumbs which turn in to a dropdown on smaller screens.
  • Beautiful fully customizable chars
  • Table with:
    • multiple selection checkboxes
    • sorter
    • user friendly zebra effect
  • Forms:
    • notification messages
    • custom input fields
    • lightweight, powerful and flexible text editor
    • dynamic submit buttons
    • custom upload field
    • datepicker
  • Extremely lightweight tabs
  • Gallery
  • Google web fonts used

Used plugins

  • Mediaqueries.js:
    a javascript build to help css and enable css3 media queries for older browsers.
    More info on the Plugin page or on the GitHub page.
  • HTML5SHIV:
    This script is the defacto way to enable use of HTML5 sectioning elements in legacy Internet Explorer, as well as default HTML5 styling in Internet Explorer 6 - 9, Safari 4.x (and iPhone 3.x), and Firefox 3.x.
    More info on the GitHub page.
  • Selectivizr:
    is a JavaScript utility that emulates CSS3 pseudo-classes and attribute selectors in Internet Explorer 6-8.
    More info on the Plugin page or on the GitHub page.
  • jScrollPane:
    is a cross-browser jQuery plugin by Kelvin Luck which converts a browser's default scrollbars into an HTML structure which can be easily skinned with CSS. Debounced and Throttled Resize Events for jQuery give us our menu a custom scroll if needed.
    More info on the Plugin page or on the smartresize GitHub page.
  • Chosen:
    is a JavaScript plugin that makes long, unwieldy select boxes much more user-friendly. It is currently available in both jQuery and Prototype flavors.
    More info on the GitHub page.
  • markItUp!:
    is a JavaScript plugin built on the jQuery library. It allows you to turn any textarea into a markup editor. Html, Textile, Wiki Syntax, Markdown, BBcode or even your own Markup system can be easily implemented. markItUp! is not meant to be a “Full-Features-Out-of-the-Box”-editor. Instead it is a very lightweight, customizable and flexible engine made to meet the developer's needs in their CMSes, blogs, forums or websites. markItUp! is not a WYSIWYG editor, and it never will be.
    More info on the Plugin page.
  • Tablesorter:
    is a jQuery plugin for turning a standard HTML table with THEAD and TBODY tags into a sortable table without page refreshes. tablesorter can successfully parse and sort many types of data including linked data in a cell. It has many useful features including:
    • Multi-column sorting;
    • Parsers for sorting text, URIs, integers, currency, floats, IP addresses, dates (ISO, long and short formats), time. Add your own easily;
    • Support secondary "hidden" sorting (e.g., maintain alphabetical sort when sorting on other criteria);
    • Extensibility via widget system;
    • Cross-browser: IE 6.0+, FF 2+, Safari 2.0+, Opera 9.0+;
    • Small code size.
    More info on the Plugin page or on the GitHub page.
  • jQuery UI Datepicker:
    jQuery UI provides interactions like Drag and Drop and widgets like Autocomplete, Tabs and Slider and makes these as easy to use as jQuery itself. Datepicker is a custom callendar plugin for easy way to pick dates and use them in any way you want.
    More info on the Plugin page.
  • Uniform:
    Uniform masks your standard form controls with custom themed controls. It works in sync with your real form elements to ensure accessibility and compatibility.
    More info on the Plugin page or on the GitHub page.
  • Lightbox2:
    Lightbox2 is a simple, unobtrusive script used to overlay images on top of the current page. It's a snap to setup and works on all modern browsers.
    More info on the Plugin page or on the GitHub page.
  • Flot charts:
    is a pure JavaScript plotting library for jQuery, with a focus on simple usage, attractive looks and interactive features.
    More info on the Plugin page or on the GitHub page.
  • bPopup:
    is a lightweight jQuery modal popup plugin. It gives you a lot of opportunities to customize so it will fit your needs.
    More info on the Plugin page or on the Plugin blog page.
